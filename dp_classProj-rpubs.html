Iris Sepal Length Distribution

Histogram Manipulator Tool

Introduction

I'd like to present a Shiny Application that uses a histogram display to help the user visualize dataset characteristics. The dataset used in the tool is the Iris dataset available in R. The distribution of Sepal Length is used to demonstrate how detail can be hidden or revealed by visualization tools such as histograms.

Tool Features:

  • User controlled bucket size
  • Reactive display of the histogram
